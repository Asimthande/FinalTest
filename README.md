Quick start

1. Clone the repository (if you haven't already):

git clone https://github.com/Asimthande/FinalTest
cd FinalTest/HotelBookingApp

2. Install dependencies:

npm install

3. Start the Expo dev server:

npm run start

4. Open the app:

- Scan the QR code with Expo Go (Android) or open with the iOS Simulator / Android emulator as prompted by Expo.
- To run directly on a platform:
  - Android: `npm run android`
  - iOS: `npm run ios` (macOS only)
  - Web: `npm run web`
